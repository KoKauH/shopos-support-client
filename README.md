# ShopOS Support

Брендированный клиент удалённой технической поддержки для пользователей **[ShopOS](https://shopos.kz)**.

Это форк замечательного проекта [RustDesk](https://github.com/rustdesk/rustdesk),
адаптированный под наш сервер `help.shopos.kz` и наш бренд.

## Зачем это нужно

Когда у магазина на ShopOS возникает проблема с кассой или админкой — клиент скачивает
программу `ShopOS Support`, запускает её и диктует свой ID + пароль специалисту техподдержки.
Специалист подключается удалённо и помогает решить проблему.

## Скачать готовый клиент

- Windows: https://help.shopos.kz/

## Что отличается от оригинального RustDesk

- Имя приложения: **ShopOS Support**
- Иконка: логотип ShopOS
- Сервер по умолчанию: `help.shopos.kz`
- Никакой пользователь не должен ничего настраивать — клиент сразу подключается к нашей инфраструктуре

Все остальные изменения зафиксированы в git-истории этого репозитория.
